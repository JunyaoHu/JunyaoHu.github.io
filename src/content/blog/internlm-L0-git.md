# 实践流程

## 破冰活动：自我介绍

每位参与者提交一份自我介绍。 提交地址：https://github.com/InternLM/Tutorial 的 camp3 分支～

命名格式为 `camp3_<id>.md`，其中 `<id>` 是您的报名问卷ID。

【大家可以叫我】内容可以是 GitHub 昵称、微信昵称或其他网名。

```
【大家可以叫我】: JunyaoHu
【坐标】：湖北
【专业/职业】：学生，计算机视觉，AIGC
【兴趣爱好】：吃饭睡觉
【项目技能】：前端
【组队情况】：solo
【本课程学习基础】：Python、CV
【本期活动目标】：卷
```

先把项目fork到自己，git clone下来

```
git clone https://github.com/JunyaoHu/Tutorial
```

文件路径应为 `./data/Git/task/`。
