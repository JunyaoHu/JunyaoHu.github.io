人脸检测

数据集WIDER FACE

  • 模型YOLOv5Face

  • Yolo5Face是深圳神目科技&LinkSprite Technologies（美国）在Yolov5模型基础上进行更新得到的人脸检测模型，在WiderFace上取得了SOTA。

  • 创新点

训练过程

文件结构

  • 在官网下载模型

  • Download WIDERFace datasets.

    Training和Validation是带标签的，test没有标签，可以暂时不下载

  • Download annotationretinaface_gt_v1.1.zip google drive.

    因为官网的训练集中人脸是不带关键点，yolo5face需要用到人脸关键点，需要下载带人脸关键点的注释。官网上给的下载地址是goole drive的地址。更换标签就完成了初步转换，要把这个初步转换的数据集放到项目里面去。

  • 验证检测精度的wider_val.txt 文件链接：wider_val.txt

    (ps: wider_val.txt only include val file names but not label information.)

  • 因为后面要做检测精度的验证，目录结构和图里面的结构保持一致。

数据准备和训练

  • 将数据集调整成YOLO5face所需要的的目录格式，命名为widerfaceyolo，用于训练。最后在widerfaceyolo文件夹中共有12879个train，3225个val。

  • 修改data/widerface.yaml文件，改为自己的数据集路径，记得把下载voc的代码注释掉，然后改成类似的格式就可以了。
  • 官网的yolo5-face.pt预训练权重加入到模型中进行训练是会报错的，用yolov5官方的yolov5s.pt权重或者其他’s’, ‘m’, ‘l’, ‘x’的作为预训练权重就可以了。放入weight文件夹

人脸识别ArcFace

数据集WebFace

CASIA-WebFace 数据集只支持学术机构的免费使用，因此，个人开发者想使用这个数据集有两个办法：

  • 看看其他使用 CASIA-WebFace 的 Github 项目，有没有放出相关的下载链接
  • （推荐）到国内资源站搜索看看，比如百度云，或者CSDN等等
  • 下载：在这找的链接，不要再花积分了，下面我给了链接
  • 百度云链接：https://pan.baidu.com/s/175Y8AB2ki2co1QPeg8Qs-w 百度云密码：5axb
